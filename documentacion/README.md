## 📄 Documentación de arquitectura

El documento [`docs/Documentacion-Arquitectura-C4.pdf`](docs/Documentacion-Arquitectura-C4.pdf) contiene el diseño completo del sistema, incluyendo:

- Descripción del sistema, actores y requerimientos funcionales/no funcionales
- Modelo de entidad-relación y modelo conceptual de datos
- Diagramas de arquitectura bajo el modelo **C4** (Contexto, Contenedores y Componentes)
- Decisiones arquitectónicas documentadas y justificadas (persistencia, comunicación entre componentes, selección de tecnologías)
- Patrones de diseño aplicados (Singleton, Facade, Middleware, Adapter)
- Trade-offs asumidos y flujos principales del sistema
